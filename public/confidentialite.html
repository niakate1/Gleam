4. Destinataires des données

Les données peuvent être transmises aux sous-traitants suivants, dans la limite nécessaire à leurs missions :

  • Supabase — hébergement de la base de données
  • Stripe — traitement des paiements
  • SendGrid / Twilio — envoi des emails transactionnels
  • Railway — hébergement du serveur applicatif
  • GitHub Pages — hébergement du site web

Les données ne sont jamais vendues à des tiers.
